title: One-line install for Code Cap
---
Code Cap picked up a release workflow so tagging a version now builds native
binaries for Linux and macOS, on both x64 and arm64, straight from GitHub
Actions. The Remotion render bundle only gets built once on Linux and is reused
across the native jobs.

Alongside that, install.sh landed so setup is a single curl-pipe-bash command
instead of cloning the repo and building locally.
