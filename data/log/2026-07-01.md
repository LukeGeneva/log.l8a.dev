date: 2026-07-01
app: Biller Lite
title: From web app to desktop app
---
Biller Lite started as a web app but this week it became a desktop app instead —
a Tauri shell wrapping the same React frontend, with data now living in a local
SQLite database instead of Supabase. That's the whole point of the app:
freelancers shouldn't need an account or a subscription just to make an invoice.
Cutting the web hosting and the Supabase dependency means there's nothing to pay
for and nothing to log into.

Along with the migration, the marketing docs and template-picker pages got
stripped out. The Supabase deploy workflow was swapped for a GitHub Actions
build that produces installers for macOS and Windows. A README now documents the
stack and how to build it.
