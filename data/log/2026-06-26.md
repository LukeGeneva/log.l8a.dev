title: Digits with repeats
---
Crack'd is a Mastermind-style daily puzzle where you guess a 4-digit secret code
in 8 tries. After each guess, you get colored feedback: green for right digit in
the right spot, yellow for right digit in the wrong spot, gray for not present
at all. Same puzzle for everyone each day, derived locally from the date via
FNV-1a hashing — no server, no account, no stored puzzle list. The whole web
version is a single HTML file you can open directly in a browser.
