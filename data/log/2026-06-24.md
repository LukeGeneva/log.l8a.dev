title: CLI tools and the distribution rabbit hole
---
Remind is a small command-line tool that creates Apple Reminders without
touching the app. You type a title and an optional time — relative durations,
absolute clock times, "tomorrow 9am" — and it lands in your default Reminders
list. The time parsing is the interesting part; everything else is a thin
wrapper around AppleScript.

The build setup took more time than the actual logic. PyInstaller bundles the
Python script into a standalone binary so there's no interpreter dependency on
the target machine. Getting that right meant using `python -m PyInstaller`
instead of the bare command, which avoids the kind of environment drift that
breaks builds silently. 

Distribution ended up being a curl-install script that pulls the latest binary
from GitHub Releases. It's one line in the README, no package manager required,
no ongoing infrastructure. The Makefile has a `release` target that tags,
builds, and uploads — so cutting a new version is a single command.
