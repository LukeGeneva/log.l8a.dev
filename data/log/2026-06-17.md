---
This site is the first thing out the door for the new dev log. It's a static
page generated by a small zero-dependency Node script (no npm install, no
framework) that reads two data files and writes one HTML file. Adding a new app
or log entry means editing a file and re-running the generator — no rebuild
pipeline, no hosted dashboard, no recurring bill.
